Školní DEMO systém

Webová aplikace slouží pro demonstraci napojení k testovací NIA pomocí IAM Keycloak protokolem OpenID Connect. Cílem je ulehčit menším OVM integraci s NIA a převést zasílané SAML atributy do srozumitelných OIDC Claims.
Vytvořené repozitáře jsou dostupné na Github.
Vytvořené Docker Images jsou zde.

Tato stránka je veřejná pro všechny.
